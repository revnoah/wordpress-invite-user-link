WordPress plugin to invite user(s) to self-onboard with a signup link.

== Description ==

WordPress plugin to invite a user or group of users to create an account on the website. Users are guided 
through a self-onboaridng process that starts with a URL. User accounts are created anonymously with 
dummy data. This allows website administrators to pre-register users before their email address, perhaps 
even their name is known. Accounts can undergo a review process before gaining access or be assigned an 
expiry date to disable the user account.

== Features ==

*   Add user(s) with anonymous details
*	Add a single user or group of users
*	Set an expiry date for user accounts
*	Require new accounts to undergo review before activation
*	Customizable templates for user-facing pages

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/invite-user-link` directory, or install the plugin through 
the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Invite User Link screen to configure the plugin.
